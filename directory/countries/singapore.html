Singapore LTA MCP Server

Active

Facilitates real-time access to Singapore's Land Transport Authority (LTA) transportation data, offering insights into bus arrivals, train services, traffic conditions, and more through integration with the LTA DataMall API.

Features
  • Get current bus arrival times by bus stop ID
  • Retrieve traffic incidents and road closures
  • Check MRT/LRT station crowding levels
  • Find parking availability in public car parks
  • Get estimated travel times on expressways

Category: Transportation

Developer: arjunkmrm

Installation
{
  "mcpServers": {
    "lta": {
      "command": "npx",
      "args": ["-y", "arjunkmrm/mcp-sg-lta"],
      "env": { 
        "LTA_API_KEY": "YOUR-API-KEY" 
      }
    }
  }
}
Note: You'll need to obtain an API key from the LTA DataMall.
Example Queries
  • "When will bus 65 arrive at stop 83139?"
  • "Are there any traffic incidents on the PIE?"
  • "How crowded is Orchard MRT station right now?"
  • "Is there parking available at Jewel Changi Airport?"
